# HR Analytics Dashboard(PowerBI)

This repository contains my **HR Analytics Dashboard** built using **Power BI**, where I explored employee attrition, demographics, job satisfaction, and workforce insights.

A key highlight of this project is implementing a **Light & Dark Mode Toggle** using custom JSON themes, improving usability and visual experience.
